# 语音热词快捷键映射配置文件
# 格式：语音指令 = 快捷键组合
# 快捷键支持：ctrl, alt, shift, win, enter, tab, space, f1-f12, 字母, 数字
# 组合键用 + 连接

# 示例：
# 打开微信 = ctrl+alt+w
# 截图 = ctrl+alt+a
# 复制 = ctrl+c
# 粘贴 = ctrl+v
